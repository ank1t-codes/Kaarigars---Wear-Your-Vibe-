# Kaarigars---Wear-Your-Vibe-
# 🛍️ KAARIGARS — Premium Fashion Website

A premium aesthetic fashion e-commerce website built for the KAARIGARS brand, combining modern UI design, smooth animations, and a seamless shopping experience inspired by Gen Z streetwear culture.

---

## Features

- Modern & aesthetic UI (Gen Z + luxury vibe)
- Smooth animations & interactive elements
- Fully responsive design (mobile + desktop)
- Clean shopping layout (Hero, New Arrivals, Popular Styles)
- Node.js backend setup
- Optimized for performance & user experience

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Styling: Custom CSS
- Assets: Custom images & graphics

## Getting Started

1. Clone the repository
git clone https://github.com/your-username/kaarigars.git

2. Install dependencies
npm install

3. Run the server
node server.js

4. Open in browser
http://localhost:3000
